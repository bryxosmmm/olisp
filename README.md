# O'lisp 

O'lisp is compiller for lisp language, written in ocaml.

## Can i do...
1) Libc functions, yes
2) Binop, yes 
3) Logicop/if/else, yes
4) functions, no
5) macros, not yet

## Examples
```lisp
(printf "Hello %s\n" "world")
```

```lisp
(if (= 1 1) (printf "true") (printf "false"))
```
